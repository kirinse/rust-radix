# Aspect Ratio

Displays content within a desired ratio.

{{#tabs global="framework" }}
{{#tab name="Leptos" }}

```toml,trunk
package = "radix-leptos-book"
features = ["aspect-ratio"]
```

{{#endtab }}
{{#endtabs }}

## Features

-   Accepts any custom ratio.

## Installation

Install the component from your command line.

{{#tabs global="framework" }}
{{#tab name="Leptos" }}

